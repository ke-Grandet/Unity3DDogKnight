## 说明
* 这是一个3D防守游戏，在三路来袭的怪物进攻中生存下来，并保护基地不被摧毁
* 视频演示：https://www.bilibili.com/video/BV1yv4y1g7rY/
* 泛型单例派生管理器类
* 通过对象池重复利用怪物对象
* 状态模式实现了玩家和怪物的角色状态机
* 范围检测和向量运算判断技能目标
* MVC分层，使用委托实现数据绑定
* 协程控制怪物进攻频率
* GL绘制小地图标注
